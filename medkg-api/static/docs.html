Introduction to MedKG

MedKG is a comprehensive medical knowledge graph that integrates information from various biomedical sources. It provides a powerful platform for researchers, healthcare professionals, and data scientists to explore and analyze complex medical relationships.

Key Features of MedKG
  • Integrated data from multiple biomedical databases
  • Advanced search capabilities
  • Interactive data visualization
  • API access for custom integrations

Searching in MedKG

MedKG offers powerful search capabilities to help you find and explore medical information efficiently.

Global Search

Global Search allows you to search the entire MedKG database, providing a comprehensive view of available information related to your query.

Global Search Features
  • Search across the entire database
  • Results displayed in two sections: Exact Matches and Partial Matches
  • Option to view further node details

Interaction Search

Interaction Search helps you find paths from a given node to a specific node type, allowing you to explore relationships within the knowledge graph.

Interaction Search Features
  • Search for paths from one node to nodes of a specific type
  • Control the search depth using the "Search Depth" parameter
  • Visualize complex relationships in the medical knowledge graph

Path Search

Path Search allows you to find the shortest paths between two different nodes in the MedKG, helping you understand the connections between various medical entities.

Path Search Features
  • Search for shortest paths between two specific nodes
  • Visualize direct and indirect relationships
  • Understand complex interactions in the medical domain
Pro Tip

Combine different search types to gain a comprehensive understanding of medical relationships. For example, use Global Search to identify entities of interest, then use Path Search to explore connections between them.

Visualising Data in MedKG

MedKG provides powerful visualization tools to help you understand and analyze complex medical relationships.

Graph Visualization

The Graph Visualization feature allows you to visually explore the relationships between different nodes in the MedKG.

Graph Visualization Features
  • Interactive node and edge exploration
  • Customizable graph layouts
  • Filtering options to focus on specific node types or relationships
  • Ability to expand and collapse node clusters

Statistical Visualizations

MedKG also offers various statistical visualizations to help you understand trends and patterns in the medical data.

Statistical Visualization Types
  • Bar charts for comparing entity frequencies
  • Network diagrams for visualizing complex interactions
  • Heat maps for identifying correlation patterns
  • Time series plots for temporal data analysis
Pro Tip

Use the combination of graph and statistical visualizations to gain both qualitative and quantitative insights into medical relationships and trends.
